dx_GFX_class: DEVICE_SetDisplayMode

DEVICE_SetDisplayMode

Cambia el modo de video en tiempo de ejecución. 

 Public Function DEVICE_SetDisplayMode(
	  Width As Long,
	  Height As Long,
	  Optional Bpp As Long = 16,
	  Optional Windowed As Boolean,
	  Optional TripleBuffer As Boolean,
	  Optional VSync As Boolean
	  Optional RefreshRate As Long = 60 ) As Boolean

Argumentos

Width
Long. Ancho en píxeles del modo de video.
Height
Long. Alto en píxeles del modo de video.
Bpp
Optional. Long. Profundidad de color del modo de video.

Default: 16

Windowed
Optional. Boolean. Indica si se inicializa en modo ventana.
TripleBuffer
Optional. Boolean. Indica si se inicializa con Triple Buffer activado.
VSync
Optional. Boolean. Indica si se inicializa con la espera de refresco vertical activada.
RefreshRate
Optional. Long. Refresco del modo de video en hertzios. Si se pasa como valor 0 se usa el valor por defecto del sistema.

Default: 60

Resultado

Boolean -  Devuelve Verdadero si se ha realizado con éxito el cambio de modo de video.

Comentarios

Asegúrese de que el modo de video exista y sea valido y soportado por el dispositivo de video, de lo contrario se producirá un fallo en el sistema de video. En caso de ocurrir se puede volver a llamar a esta función para aplicar un modo de video correcto, por lo tanto no debe preocuparse por los datos cargados en memoria.

En modo ventana se puede establecer cualquier resolución. En modo a pantalla completa solo se aceptan los modos de video soportados por el dispositivo de video.
