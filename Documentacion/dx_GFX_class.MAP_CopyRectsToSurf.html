MAP_CopyRectsToSurf

Copia una serie de regiones de un grafico en una superficie. 

 Public Function MAP_CopyRectsToSurf(
	  SrcMap As Long,
	  DestSurf As Long,
	  SrcRect() As GFX_Rect,
	  DestPixel() As Vertex ) As Boolean

Argumentos

SrcMap
Long. Identificador del grafico origen.
DestSurf
Long. Identificador de la superficie destino.
SrcRect
GFX_Rect. Lista de regiones que se copiaran del grafico origen a la superficie destino. El argumento se debe pasar apuntando al primer vector del array, el 0.
DestPixel
Vertex. Lista de coordenadas donde se copiaran las regiones en la superficie destino. El argumento se debe pasar apuntando al primer vector del array, el 0.

Resultado

Boolean -  Devuelve Verdadero si la operación se ha realizado con éxito.

Comentarios

No se respeta el ColorKey de los gráficos en esta operación.
