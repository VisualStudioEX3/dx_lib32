SURF_GetPixel

Lee un píxel de una lista de píxeles en memoria. 

 Public Function SURF_GetPixel(
	  Id As Long,
	  X As Long,
	  Y As Long ) As Long

Argumentos

Id
Long. Identificador de la lista de píxeles de la superficie.
X
Long. Coordenada horizontal de lectura.
Y
Long. Coordenada vertical de lectura.

Resultado

Long -  Devuelve el valor ARGB del píxel leído.

Comentarios

Para poder leer un píxel de una superficie primero se ha de crear una lista de píxeles con los valores de los píxeles de la superficie original.
