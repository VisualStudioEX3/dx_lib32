dx_Sound_Class: SOUND_Load

SOUND_Load

Carga un archivo de audio desde disco como un efecto de sonido. 

 Public Function SOUND_Load(
	  Filename As String ) As Long

Argumentos

Filename
String. Nombre y ruta del archivo de sonido.

Resultado

Long -  Devuelve el código identificador para el sonido en memoria, un valor comprendido entre 0 y el máximo permitido por el tipo Long. Si no se puede cargar el sonido se devuelve SND_FILENOTLOAD.

Comentarios

Esta función solo carga archivos de ondas (WAVE) *.WAV. Se pueden cargar tantos efectos de sonido como se deseen teniendo como limite la memoria del sistema.
