dx_Video_Class: VIDEO_Load

VIDEO_Load

Carga un archivo de video desde disco como una secuencia de video. 

 Public Function VIDEO_Load(
	  Filename As String ) As Long

Argumentos

Filename
String. Nombre y ruta del archivo de video.

Resultado

Long -  Devuelve el código identificador para la secuencia de video en memoria, un valor comprendido entre 0 y el máximo permitido por el tipo Long. Si no se puede cargar la secuencia de video se devuelve -1.

Comentarios

Esta función soporta de forma nativa formatos *.AVI y *.MPG (MPEG1 y 2). Se pueden procesar otros formatos de audio como el *.WMV o DivX si se instalan los codecs necesarios para DirectShow. Se pueden cargar tantas secuencias de video como se deseen teniendo como limite la memoria del sistema.
